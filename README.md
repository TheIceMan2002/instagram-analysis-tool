# instagram-analysis-tool

A Python script that parses Instagram data exports to identify non-reciprocal follower relationships.

## Features
- Parses Instagram JSON data exports
- Handles multiple data format variations and encoding issues
- User-friendly CLI with file export functionality

## How to Run
1. Download your Instagram data export from Instagram's settings
2. Install dependencies: `pip install json`
3. Run the script: `python check_followers.py`
